# 🎵 Spotify Clone — Frontend

A Spotify-inspired music streaming interface built with React and Vite — featuring audio playback, playlist browsing, and a fully responsive player UI.

## ✨ Features

- **Audio Playback** — Play, pause, skip, and seek tracks with a persistent bottom player bar
- **Playlist Browsing** — Browse albums and playlists with a sidebar navigation
- **Now Playing UI** — Dynamic album art, track info, and progress bar
- **Responsive Design** — Fully responsive layout built with Tailwind CSS
- **Fast Dev Experience** — Powered by Vite with Hot Module Replacement

---

## 🛠️ Tech Stack

React · Vite · Tailwind CSS · JavaScript

## 🚀 Getting Started

```bash
git clone https://github.com/mohamed-ayad40/Spotify-Clone-3.git
cd Spotify-Clone-3
npm install
npm run dev
```
